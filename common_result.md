3. Key Technical Concepts:

- xUnit: тестовый фреймворк для написания unit-тестов
- FluentAssertions: библиотека для написания читаемых утверждений в тестах
- AutoFixture: библиотека для автоматической генерации тестовых данных
- SOLID: принципы объектно-ориентированного дизайна
- ASP.NET Core Web API: платформа для создания RESTful API
- Controllers: компоненты для обработки HTTP-запросов
- Repositories: паттерн для абстракции доступа к данным
