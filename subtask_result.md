# Отчет о выполнении подзадачи 1.3

## Подзадача 1.3: Реализовать логику генерации промокода

### Промт
Реализовать метод для создания уникального номера промокода с заданным форматом ([A-Z]{4}-[0-9]{6}), генерировать случайную скидку (10-50%) и устанавливать срок действия (30 дней от текущей даты).

### Выполненные действия

1. Добавлены using директивы для System.Security.Cryptography и System.Text
2. Добавлены XML-комментарии для класса PromoCode и всех его свойств
3. Реализован статический метод GenerateCode() для генерации уникального кода промокода в формате [A-Z]{4}-[0-9]{6}
4. Реализован статический метод GenerateDiscount() для генерации случайной скидки от 10% до 50%
5. Реализован статический метод Create() для создания нового промокода с заданными параметрами и автоматической генерацией кода, скидки и сроков действия
6. Добавлены проверки входных параметров в метод Create() с помощью ArgumentNullException
7. Проверка компиляции проекта (dotnet build), которая прошла успешно

### Результат
Код компилируется без ошибок. Добавлены необходимые методы для генерации промокодов с требуемым форматом и параметрами.
